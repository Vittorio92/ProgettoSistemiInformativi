# ProgettoSistemiInformativi
Progetto per il corso di Sistemi Informativi — A.A. 2025/2026. Proof of Time consiste nello studio di fattibilità  per una piattaforma innovativa pensata per il settore degli orologi di lusso, con l’obiettivo di garantire:  Autenticità  - Tracciabilità   - Trasparenza   - Riduzione della contraffazione   - Valorizzazione del mercato dell’usato
